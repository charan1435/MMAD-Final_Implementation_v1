What is Adversarial MRI Defense?

Adversarial attacks on medical imaging can pose significant risks to diagnostic accuracy and patient safety. Our system provides tools to detect such attacks, purify affected images, and study attack methods to build better defenses.

This platform integrates state-of-the-art deep learning models to address three key challenges in securing MRI imaging:

  • Detection - Identify whether an MRI image has been adversarially manipulated
  • Purification - Remove adversarial perturbations while preserving diagnostic features
  • Attack Simulation - Generate adversarial examples to test defenses

Technical Details

Detection Model

Our classifier uses an ensemble of CNN and Vision Transformer architectures with attention mechanisms to accurately detect various adversarial attacks including FGSM, BIM, and PGD.

Purification Model

The purification model employs a U-Net architecture enhanced with self-attention and trained using a two-phase approach with GAN-based refinement for superior image quality.

Attack Methods

Our system can generate adversarial examples using Fast Gradient Sign Method (FGSM), Basic Iterative Method (BIM), and Projected Gradient Descent (PGD) attacks.
